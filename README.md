# Introduction
This is the repository for the manuscript, **Effects of preservatives on the recovery of environmental DNA (eDNA) from seawater: Implications for field sampling in remote areas** (Labrador et al., submitted to *Molecular Ecology Resources*).

This repository contains a reproducible R-based analysis pipeline for processing and modeling DNA quantification and amplification data. The workflow integrates data wrangling, visualization, and statistical modeling steps to evaluate the effects of preservation and extraction method on eDNA yield and amplification success. The pipeline is organized into modular scripts that can be executed sequentially, from raw data compilation to model-based inferences.

---

# Directory Structure

* `scripts` - all instructions for data manipulation
* `data` - all raw and processed data files
* `results` - statistica results and figures generated from R scripts 

---

# Analytical Pipeline

Check [`./scripts/`](./scripts/) subdirectory for the full details.

In summary, the analytical pipeline is as follows:

| Step | Script | Brief Description |
| ---- | ------ | ----------------- |
|  01  |[`01_wrangle_quant-results.R`](./01_wrangle_quant-results.R)|compiles and standardizes DNA quantification data generated by the [DNA Quantification Toolkit](https://github.com/tamucc-gcl/gcl_bioinformatic_tools/blob/main/labTools/DNA_Quant/README.md); merges all quantification results into a single, clean dataset ready for downstream statistical analysis and visualization
|  02  |[`02_wrangle_data-frame-for-statistical-tests.R`](02_wrangle_data-frame-for-statistical-tests.R)|prepares and consolidates all data required for downstream analyses
|  03  | [`03_plot_timeline.R`](./03_plot_timeline.R) | visualizes the sample preservation timeline for eDNA field collections and and the subsequent coldchain events.
|  04  | [`04_plot_collection-sites.R`](./04_plot_collection-sites.R) |  generates a map showing sampling locations in Rota Island, Commonwealth of the Northern Mariana Islands (CNMI)
|  05  |[`05_model_quant-results.R`](./05_model_quant-results.R) | fits a GLMM on DNA concentration
|  06  | [`06_model_amplification-yield.R`](./06_model_amplification-yield.R) | fits a GLMM on PCR amplification yield
